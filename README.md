## The first step to installing GRAX is to deploy your Heroku environment. Once complete this your Salesforce Instance will have to be configured. 

### 1. Deploy GRAX Enterprise Application to Heroku 

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/HardingPoint/GRAXEnterprise/blob/master)
 
 1. Right click in new tab the `Deploy` button above to begin the GRAX Enterprise deployment to your Heroku instance
 2. Specify your App name
 3. Deploy App
 
### 2. Salesforce Configuration

# To configure GRAX, please use the following link: <a href="https://github.com/HardingPoint/GRAXForSFDC" target="_new">GRAXForSFDC</a>
